# DetectiveSAM Checkpoints

This GitHub repo does not bundle the `.pth` checkpoint files.

The full runnable bundle is hosted on Hugging Face:

- https://huggingface.co/Gertlek/DetectiveSAM

To run inference from this GitHub checkout, download these files from Hugging Face into this directory:

- `model_epoch22_batch999_score1.1114.pth`
- `detective_sam_sota.pth`

The small YAML sidecars in this directory are kept in Git so the checkpoint aliases continue to resolve once the `.pth` files are present locally.
